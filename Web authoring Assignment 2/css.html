Evolution of CSS

Introduction to CSS

The creation of HTML and the World Wide Web Consortium led to the creation of CSS. CSS has had many different versions over the years but the most important ones are;

    CSS 1 CSS 2 CSS 2.1 CSS 3

CSS 1

The first version of CSS was created on 17th November 1996. It was created to return the roots of the internet and get rid of all the content from the browser. The style sheet was created to separate the design from the default coding. It increased the speed of browsers, made the web page coding easier to read. The author can easily access the mistakes they may make in the styling of the webpage with it separate from the information.

CSS 2

This evolved the CSS 1 into a multi-functional page with the style sheet now able to make the web page be seen in braille, work with and format printers to your style. It also tailored webpages for handheld devices such as phones and aural device for the hearing impaired.

CSS 2.1

This version of CSS was created in June 2001. It was created because of errors reported in CSS 2 and the World Wide Web Consortium wanted to implement a small amount of change to the way we code by taking out things that will be obsolete with CSS 3.0.

CSS 3

CSS 3 was released in 2011 and 2012. It added over 50 new modules such as Color, Media Queries, Grid Layout and transitions. It also added several new selectors, values and properties to the CSS database.
